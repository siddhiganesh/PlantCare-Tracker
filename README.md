🌿 Plant Care Tracker
A full-stack Plant Care Tracker built with Flask, Vue 3, TypeScript,
Pinia and SQLite.
Features
JWT Authentication
Plant CRUD
Image Uploads
Automatic Watering Reminders
Watering History
Dashboard
Profile Management
Tech Stack
Frontend
Vue 3
TypeScript
Vite
Pinia
Bootstrap 5
Axios
Backend
Flask
SQLAlchemy
Flask-JWT-Extended
Flask-Migrate
Flask-CORS
Database
SQLite
Installation
Backend
``` bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
Frontend
``` bash
cd frontend
npm install
npm run dev
```
